Install BM32 OS

1. Download Project

Download the full BM32 project archive (.zip file).

Download ZIP

2. Extract Archive

Unpack the ZIP file to any folder on your computer.

3. Connect your CYD

4. Launch install.ps1

On Windows: install.ps1

On Linux: install.sh

4. Done!
